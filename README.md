## Getting Started

### Installation

1. Clone the repo
    ```sh
    git clone git@github.com:zhafirahajar/FP3_H8.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Database Migrate
    ```sh
    npx sequelize db:migrate
    ```
4. Start the server
    ```sh
    node app
    ```
    or
    ```sh
    nodemon app
    ```
